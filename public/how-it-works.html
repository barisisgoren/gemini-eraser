How It Works

Last updated: April 16, 2026

The app estimates Gemini's visible watermark area and reverses the blend locally instead of sending your image anywhere.

Genel Mantık

Araç, Gemini tarafında görünen filigranın konumunu ve boyutunu tahmin eder. Ardından bu görünür katmanı matematiksel olarak tersine çevirerek sonucu olabildiğince temiz bir biçimde yeniden oluşturur. Bu yaklaşım klasik sunucu tabanlı düzeltmeler yerine doğrudan tarayıcıda çalışır.

1. Detect the expected watermark area

The app uses Gemini sizing and placement rules to find the most likely visible watermark region.

2. Reverse the visible blend

Because the visible overlay pattern is known, the source pixels can be reconstructed mathematically instead of painted over.

Formula

watermarked = alpha * logo + (1 - alpha) * original

This equation represents the visible watermarked image.

Formula

original = (watermarked - alpha * logo) / (1 - alpha)

This inverse equation is applied channel by channel to restore the original pixels.

3. Apply Gemini sizing rules

The remover switches between calibrated watermark templates and margins before any adaptive refinement is attempted.

Condition Watermark size Right margin Bottom margin
Width > 1024 and height > 1024 96 x 96 64px 64px
All other supported sizes 48 x 48 32px 32px

4. Keep processing local

Once the page is loaded, the worker, styles, and locale files run locally so the image stays on the device.
